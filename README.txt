# SalesReportDE101
A simple data application to get insightful queries on a sample sales report dataset, using Java REST APIs, AWS data engineering and analytics services.
The application ingests basic analytical queries from AWS Athena onto a quarterly sales report AWS Glue table. 
The application has multiple REST endpoints exposed to respective views.
<br/>
How to run:
<br/>
Run the App.java class
<br/>
Enter the URL for the respective view on a browser
